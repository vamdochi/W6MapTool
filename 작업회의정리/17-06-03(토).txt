
--------------- Map ---------------
1. 맵은 n * n 을 진행 ( n Size는 변경 가능 )
2. 맵은 작은 파츠로 조립됨 ( 제작은 작은 파츠로 여러개 제작 후 조립함 )
3. 최상위 오브젝트에 어떠한 조건이 존재시 그에 해당하는 최상위 오브젝트가 존재함. ( 화염병 오브젝트가 존재시 길을 막는 나무벽이 존재함 )

--------------- Battle ---------------
1. 타일 기본 조작( Move / Attack ) 부터 개발
2. Input Controller 의논 
	1) 각 Side별로 방향키중 2개 분배
	2) 왼쪽 or 오른쪽 방향키 4개있는 컨트롤러 세팅
3. Guard System 의논 
	 -> 가드상태는 몬스터의 공격을 막고 카운터 어택이 존재함
	1) 방향키 두번 눌렀을떄 해당 방향 방어
	2) 가드 버튼을 따로 둔다 -> ( 4방향? 원버튼? 두개의 스타일은 다름 )
